# Семейный бюджет

Простой веб-сайт для учёта семейных доходов и расходов. Без регистрации и серверов — все данные сохраняются в браузере (`localStorage`).

## Возможности

- Добавление доходов и расходов с категориями
- Сводка: доходы, расходы, остаток, процент расходов от дохода
- Прогресс-бар использования дохода
- Фильтрация и сортировка истории операций
- Экспорт и импорт данных в JSON
- Светлая и тёмная тема
- Приветственный лендинг с описанием сервиса

## Запуск

Откройте `index.html` в браузере — двойной клик или через локальный сервер:

```bash
# Python 3
python3 -m http.server 8080

# или Node.js (npx)
npx serve .
```

После запуска откройте в браузере: `http://localhost:8080`

## Данные

| Ключ в localStorage   | Описание              |
|-----------------------|-----------------------|
| `familyBudgetData`    | Список операций       |
| `familyBudgetTheme`   | Тема: `light` / `dark`|

Данные не синхронизируются между устройствами автоматически. Для переноса используйте экспорт/импорт JSON.
